# detector-fake-news.
Detector de Fake News es un minijuego web estilo arcade retro CRT que pone a prueba tu habilidad para distinguir noticias verdaderas de las falsas, todo mientras disfrutás de una experiencia rápida y adictiva. Con estética pixelada en rojo, blanco y negro, tipografía inspirada en minijuegos de los años 80 y efectos visuales que evocan las clásicas pantallas antiguas, este juego combina diversión y aprendizaje de manera única.

Creado especialmente para la página web de Ccomusoc con fines educativos, Detector de Fake News transforma la detección de información confiable en un desafío interactivo y entretenido, ideal para entrenar tu pensamiento crítico sin siquiera darte cuenta.
