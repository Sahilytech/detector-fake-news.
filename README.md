# detector-fake-news.
Detector de Fake News es un minijuego web estilo arcade retro CRT, diseñado para entrenar tu capacidad de reconocer noticias falsas mientras jugás. Con estética pixel roja/blanca/negra, tipografía tipo Minecraft y efectos visuales de pantalla antigua, el juego mezcla educación digital con una experiencia rápida y adictiva.
